# DevOps Demo Platform (Course Repo)

This repository is built during a 140-day Junior DevOps course.
It contains infrastructure artifacts, CI/CD pipelines, IaC, Kubernetes manifests, and documentation.

## Structure
- docs/ — runbooks, architecture, interview kit
- scripts/ — bash utilities
- compose/ — local Docker Compose environment
- k8s/ — Kubernetes manifests
- helm/ — Helm charts
- terraform/ — AWS IaC
- ansible/ — server bootstrap automation

“Prerequisites: terminal + Git”
