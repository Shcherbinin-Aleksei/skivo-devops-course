## Day 2
## Navigation
- pwd — show current directory
- ls / ls -la — list files (including hidden)
- cd <dir> — go to directory
- cd .. — up
- cd ~ — home
- cd - — previous directory

## Create
- mkdir -p path — create folders
- touch file — create empty file

## Copy/Move/Delete
- cp src dst — copy
- mv src dst — move/rename
- rm file — delete file (no trash)
