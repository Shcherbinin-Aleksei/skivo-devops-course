# Week 1 Checklist — Linux basics

- [ ] I can navigate: pwd / ls -la / cd / cd .. / cd ~ / cd -
- [ ] I can create & manage files: mkdir -p / touch / cp / mv / rm (carefully)
- [ ] I can view & search logs: less / head / tail / tail -f / grep
- [ ] I can edit files on server: nano (save/exit/search), vi (i, Esc, :wq, :q!)
- [ ] I understand permissions: user/group/others, r/w/x; I avoid 777
- [ ] I can install packages (Ubuntu/WSL): apt update / apt install / apt show / apt search
- [ ] Repo docs are readable: docs/README.md and docs/quickstart-week1.md
